# `daft repo remove` Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use
> superpowers:subagent-driven-development (recommended) or
> superpowers:executing-plans to implement this plan task-by-task. Steps use
> checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement `daft repo remove [<path>]`, a top-level command that
removes a Git repository — bare directory plus all checked-out worktrees —
running `worktree-pre-remove` and `worktree-post-remove` lifecycle hooks per
worktree. Add a sandbox `clean-repos` script that uses the new command to clean
up test repos.

**Spec:** `docs/superpowers/specs/2026-04-28-remove-repo-design.md`

**Architecture:** Reuse the existing `OperationTable` renderer plus the
`SyncDag` / `DagExecutor` infrastructure that powers `daft prune`. Two new
`TaskId` variants (`RemoveWorktree`, `RemoveBare`) and a new
`SyncDag::build_remove_repo` constructor; per-worktree tasks run in parallel; a
terminal `RemoveBare` task depends on every worktree task. Hook failures never
abort — they are recorded and shown in the post-run summary.

**Tech Stack:** Rust 2021, clap 4 (derive), anyhow, ratatui (via daft's existing
`OperationTable`), bash for the sandbox helper, YAML manual scenarios for
end-to-end tests.

/// Build a DAG for `daft repo remove`.
///
/// Each `(branch_name, worktree_path)` becomes a `RemoveWorktree` task with no
/// inter-task dependencies (parallel removal). A terminal `RemoveBare` task
/// depends on every worktree task; if `worktrees` is empty, `RemoveBare` is
/// the sole task with no dependencies.
pub fn build_remove_repo(
    worktrees: Vec<(String, std::path::PathBuf)>,
    bare_git_dir: std::path::PathBuf,
) -> Self {
    let mut tasks = Vec::new();
    let mut dependencies: Vec<Vec<usize>> = Vec::new();
    let mut dependents: Vec<Vec<usize>> = Vec::new();

    let mut push_task = |task: SyncTask, deps: Vec<usize>| -> usize {
        let idx = tasks.len();
        tasks.push(task);
        for &dep in &deps {
            if dependents.len() <= dep {
                dependents.resize_with(dep + 1, Vec::new);
            }
            dependents[dep].push(idx);
        }
        dependencies.push(deps);
        if dependents.len() <= idx {
            dependents.resize_with(idx + 1, Vec::new);
        }
        idx
    };

    let mut worktree_indices = Vec::new();
    for (branch, path) in &worktrees {
        let idx = push_task(
            SyncTask {
                id: TaskId::RemoveWorktree(path.clone()),
                phase: OperationPhase::RemoveRepo,
                worktree_path: Some(path.clone()),
                branch_name: branch.clone(),
            },
            vec![],
        );
        worktree_indices.push(idx);
    }

    push_task(
        SyncTask {
            id: TaskId::RemoveBare,
            phase: OperationPhase::RemoveRepo,
            worktree_path: Some(bare_git_dir),
            branch_name: "(bare)".to_string(),
        },
        worktree_indices,
    );

    Self {
        tasks,
        dependencies,
        dependents,
        rebase_branch: None,
        push: false,
    }
}
```

//! Core logic for `daft repo remove`.
//!
//! Resolves a repo target from a path or cwd, enumerates its worktrees,
//! and provides the per-task execution helpers used by the command.

use anyhow::{anyhow, bail, Context, Result};
use std::path::{Path, PathBuf};

use crate::git::GitCommand;

/// Identity of a repo to be removed.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct RepoTarget {
    /// The bare git directory (resolved via `git rev-parse --git-common-dir`).
    pub bare_git_dir: PathBuf,
    /// The directory immediately containing the bare git dir (project root).
    pub project_root: PathBuf,
}

/// Resolve a repo target from an optional user-supplied path.
///
/// If `path` is `None`, the current working directory is used. Errors with a
/// clear message if the path is not inside any git repository.
pub fn resolve_repo(path: Option<&Path>) -> Result<RepoTarget> {
    let start = match path {
        Some(p) => p.to_path_buf(),
        None => std::env::current_dir().context("could not determine current directory")?,
    };
    if !start.exists() {
        bail!("{}: no such file or directory", start.display());
    }

    let git = GitCommand::new(true);
    let common_dir = git
        .rev_parse_in("--git-common-dir", &start)
        .map_err(|_| anyhow!("{} is not inside a Git repository", start.display()))?;

    let bare_git_dir = std::fs::canonicalize(&common_dir)
        .with_context(|| format!("could not canonicalize {}", common_dir.display()))?;

    let project_root = bare_git_dir
        .parent()
        .ok_or_else(|| anyhow!("git dir {} has no parent", bare_git_dir.display()))?
        .to_path_buf();

    Ok(RepoTarget {
        bare_git_dir,
        project_root,
    })
}
```

```rust
/// A single checked-out worktree of the target repo.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct WorktreeEntry {
    pub path: PathBuf,
    pub branch: Option<String>,
    pub is_bare: bool,
    pub is_detached: bool,
}

/// Enumerate every worktree of `target` via `git worktree list --porcelain`.
///
/// The bare repo entry (the one with `bare`) is filtered out — `RemoveBare`
/// covers that — and only checked-out worktrees are returned.
pub fn enumerate_worktrees(target: &RepoTarget) -> Result<Vec<WorktreeEntry>> {
    let output = std::process::Command::new("git")
        .arg("--git-dir")
        .arg(&target.bare_git_dir)
        .args(["worktree", "list", "--porcelain"])
        .output()
        .context("git worktree list failed")?;
    if !output.status.success() {
        bail!(
            "git worktree list exited {}",
            output.status.code().unwrap_or(-1)
        );
    }
    let stdout = String::from_utf8(output.stdout).context("worktree-list output not UTF-8")?;

    let mut out = Vec::new();
    let mut path: Option<PathBuf> = None;
    let mut branch: Option<String> = None;
    let mut is_bare = false;
    let mut is_detached = false;

    for line in stdout.lines() {
        if line.is_empty() {
            if let Some(p) = path.take() {
                if !is_bare {
                    out.push(WorktreeEntry {
                        path: p,
                        branch: branch.take(),
                        is_bare,
                        is_detached,
                    });
                }
                branch = None;
                is_bare = false;
                is_detached = false;
            }
            continue;
        }
        if let Some(rest) = line.strip_prefix("worktree ") {
            path = Some(PathBuf::from(rest));
        } else if let Some(rest) = line.strip_prefix("branch refs/heads/") {
            branch = Some(rest.to_string());
        } else if line == "bare" {
            is_bare = true;
        } else if line == "detached" {
            is_detached = true;
        }
    }
    // Handle final entry without trailing blank line.
    if let Some(p) = path {
        if !is_bare {
            out.push(WorktreeEntry {
                path: p,
                branch,
                is_bare,
                is_detached,
            });
        }
    }
    Ok(out)
}
```

/// Outcome of removing a single worktree.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum RemoveWorktreeOutcome {
    /// `git worktree remove --force` succeeded.
    Removed,
    /// Fell back to `rm -rf` after `git worktree remove` failed.
    RemovedViaFallback,
}

/// Best-effort filesystem removal of a single worktree.
///
/// 1. `git worktree remove --force <path>`
/// 2. On failure: log to stderr, then `rm -rf <path>` and prune git's stale
///    administrative entries.
/// 3. Errors only when both attempts fail.
pub fn remove_worktree_filesystem(
    target: &RepoTarget,
    worktree_path: &Path,
) -> Result<RemoveWorktreeOutcome> {
    let try_git_remove = std::process::Command::new("git")
        .arg("--git-dir")
        .arg(&target.bare_git_dir)
        .args(["worktree", "remove", "--force"])
        .arg(worktree_path)
        .output()
        .context("git worktree remove failed to launch")?;

    if try_git_remove.status.success() && !worktree_path.exists() {
        return Ok(RemoveWorktreeOutcome::Removed);
    }

    // Fallback: rm -rf + prune the stale entry.
    if worktree_path.exists() {
        std::fs::remove_dir_all(worktree_path)
            .with_context(|| format!("rm -rf {} failed", worktree_path.display()))?;
    }
    let _ = std::process::Command::new("git")
        .arg("--git-dir")
        .arg(&target.bare_git_dir)
        .args(["worktree", "prune"])
        .status();
    Ok(RemoveWorktreeOutcome::RemovedViaFallback)
}
```

/// Remove the bare git directory and any now-empty ancestor directories
/// up to (but not including) the first non-empty one. Drops the trust DB
/// entry for the bare path.
pub fn remove_bare_directory(target: &RepoTarget) -> Result<()> {
    if target.bare_git_dir.exists() {
        std::fs::remove_dir_all(&target.bare_git_dir).with_context(|| {
            format!("rm -rf {} failed", target.bare_git_dir.display())
        })?;
    }
    // Walk upward, stopping at the first non-empty dir. This handles
    // contained / sibling / nested layouts uniformly.
    let mut cursor = target.project_root.clone();
    while cursor.exists() {
        let mut iter = match std::fs::read_dir(&cursor) {
            Ok(it) => it,
            Err(_) => break,
        };
        if iter.next().is_some() {
            break;
        }
        if std::fs::remove_dir(&cursor).is_err() {
            break;
        }
        match cursor.parent() {
            Some(p) => cursor = p.to_path_buf(),
            None => break,
        }
    }

    // Drop trust DB entry. Best-effort: we never want a missing trust file
    // to abort removal that already completed on the filesystem.
    if let Ok(mut db) = crate::hooks::TrustDatabase::load() {
        if db.reset_repo(&target.bare_git_dir) {
            let _ = db.save();
        }
    }
    Ok(())
}

use crate::core::worktree::remove_repo::{
    self, RemoveWorktreeOutcome, RepoTarget, WorktreeEntry,

use crate::hooks::{HookContext, HookExecutor, HookType, RemovalReason};
use std::time::Instant;

/// Execute one `RemoveWorktree` task. Hook failures never abort.
pub fn execute_remove_worktree_task(
    target: &RepoTarget,
    entry: &WorktreeEntry,
    hooks_config: &crate::hooks::HooksConfig,
    tx: &mpsc::Sender<DagEvent>,
) -> (TaskStatus, TaskMessage) {
    let label = entry.branch.clone().unwrap_or_else(|| "(detached)".into());

    // 1. worktree-pre-remove
    run_hook_best_effort(
        target, entry, HookType::PreRemove, hooks_config, tx, &label,
    );

    // 2. Filesystem removal
    if let Err(e) = remove_repo::remove_worktree_filesystem(target, &entry.path) {
        return (
            TaskStatus::Failed,
            TaskMessage::Failed(format!("remove failed: {e}")),
        );
    }

    // 3. worktree-post-remove
    run_hook_best_effort(
        target, entry, HookType::PostRemove, hooks_config, tx, &label,
    );

    (TaskStatus::Succeeded, TaskMessage::Removed)
}

fn run_hook_best_effort(
    target: &RepoTarget,
    entry: &WorktreeEntry,
    hook_type: HookType,
    hooks_config: &crate::hooks::HooksConfig,
    tx: &mpsc::Sender<DagEvent>,
    label: &str,
) {
    let executor = match HookExecutor::new(hooks_config.clone()) {
        Ok(e) => e,
        Err(_) => return,
    };
    let started = Instant::now();
    let _ = tx.send(DagEvent::HookStarted {
        branch_name: label.to_string(),
        hook_type,
    });
    let ctx = HookContext::for_worktree_removal(
        &target.bare_git_dir,
        &entry.path,
        entry.branch.as_deref(),
        RemovalReason::ManualRemove,
    );
    let result = executor.run(hook_type, &ctx);
    let elapsed = started.elapsed();
    let (success, warned, exit_code, output) = match result {
        Ok(r) => (r.success, r.warned, r.exit_code, r.output),
        Err(e) => (false, false, None, Some(format!("{e}"))),
    };
    let _ = tx.send(DagEvent::HookCompleted {
        branch_name: label.to_string(),
        hook_type,
        success,
        warned,
        duration: elapsed,
        exit_code,
        output,
    });
}

/// Execute the terminal `RemoveBare` task.
pub fn execute_remove_bare_task(target: &RepoTarget) -> (TaskStatus, TaskMessage) {
    match remove_repo::remove_bare_directory(target) {
        Ok(()) => (TaskStatus::Succeeded, TaskMessage::Removed),
        Err(e) => (
            TaskStatus::Failed,
            TaskMessage::Failed(format!("bare removal failed: {e}")),
        ),
    }
}

fn run_sequential(
    target: &crate::core::worktree::remove_repo::RepoTarget,
    worktrees: &[crate::core::worktree::remove_repo::WorktreeEntry],
) -> Result<()> {
    use crate::commands::sync_shared::{
        execute_remove_bare_task, execute_remove_worktree_task,
    };
    let hooks_config = crate::hooks::HooksConfig::default();
    // Channel only used by the helpers' DagEvent emitters; we drain it inline.
    let (tx, rx) = std::sync::mpsc::channel();

    let mut hook_summaries = Vec::new();
    let mut any_failed = false;
    for entry in worktrees {
        let label = entry.branch.clone().unwrap_or_else(|| "(detached)".into());
        let (status, msg) = execute_remove_worktree_task(target, entry, &hooks_config, &tx);
        println!(
            "  {label}: {}",
            match msg {
                crate::core::worktree::sync_dag::TaskMessage::Removed => "removed",
                crate::core::worktree::sync_dag::TaskMessage::Failed(ref e) => {
                    any_failed = true;
                    e.as_str()
                }
                _ => "removed",
            }
        );
        if matches!(status, crate::core::worktree::sync_dag::TaskStatus::Failed) {
            any_failed = true;
        }
        // Drain hook events for the summary.
        while let Ok(ev) = rx.try_recv() {
            if let crate::core::worktree::sync_dag::DagEvent::HookCompleted {
                branch_name, hook_type, success, warned, duration, exit_code, output,
            } = ev
            {
                if !success || warned {
                    hook_summaries.push(HookSummary {
                        branch_name, hook_type, success, warned, duration, exit_code, output,
                    });
                }
            }
        }
    }

    let (bare_status, bare_msg) = execute_remove_bare_task(target);
    println!(
        "  (bare): {}",
        match bare_msg {
            crate::core::worktree::sync_dag::TaskMessage::Removed => "removed",
            crate::core::worktree::sync_dag::TaskMessage::Failed(ref e) => e.as_str(),
            _ => "removed",
        }
    );
    if matches!(bare_status, crate::core::worktree::sync_dag::TaskStatus::Failed) {
        any_failed = true;
    }

    print_hook_summary(&hook_summaries);
    if any_failed {
        bail!("repo removal had failures (see above)");
    }
    Ok(())
}

struct HookSummary {
    branch_name: String,
    hook_type: crate::hooks::HookType,

fn run_tui(
    target: &crate::core::worktree::remove_repo::RepoTarget,
    worktrees: &[crate::core::worktree::remove_repo::WorktreeEntry],
) -> Result<()> {
    use crate::commands::sync_shared::{
        execute_remove_bare_task, execute_remove_worktree_task,
    };
    use crate::core::worktree::list::Stat;
    use crate::core::worktree::sync_dag::{
        DagEvent, DagExecutor, OperationPhase, SyncDag, SyncTask, TaskId, TaskMessage,
        TaskOutcome, TaskStatus,
    };
    use crate::output::tui::operation_table::{OperationTable, TableConfig};
    use std::collections::HashSet;
    use std::path::PathBuf;
    use std::sync::Arc;

    let phases = vec![OperationPhase::RemoveRepo];

    // Build a WorktreeInfo seed list — one row per worktree plus one
    // synthetic "(bare)" row.
    let worktree_infos = build_seed_rows(target, worktrees);

    let dag = SyncDag::build_remove_repo(
        worktrees
            .iter()
            .map(|w| (w.branch.clone().unwrap_or_else(|| "(detached)".into()), w.path.clone()))
            .collect(),
        target.bare_git_dir.clone(),
    );

    let (tx, rx) = std::sync::mpsc::channel();
    let target_arc = Arc::new(target.clone());
    let entries_arc: Arc<Vec<_>> = Arc::new(worktrees.to_vec());
    let hooks_config = crate::hooks::HooksConfig::default();
    let hooks_arc = Arc::new(hooks_config);

    let tx_for_tasks = tx.clone();
    let target_for_tasks = Arc::clone(&target_arc);
    let entries_for_tasks = Arc::clone(&entries_arc);
    let hooks_for_tasks = Arc::clone(&hooks_arc);

    let orchestrator = std::thread::spawn(move || {
        let executor = DagExecutor::new(dag, tx);
        executor.run(
            move |task: &SyncTask, outcomes: &HashSet<TaskOutcome>|
                  -> (TaskStatus, TaskMessage, HashSet<TaskOutcome>) {
                match &task.id {
                    TaskId::RemoveWorktree(path) => {
                        let entry = entries_for_tasks
                            .iter()
                            .find(|e| &e.path == path)
                            .cloned()
                            .expect("entry for task path");
                        let (s, m) = execute_remove_worktree_task(
                            &target_for_tasks, &entry, &hooks_for_tasks, &tx_for_tasks,
                        );
                        (s, m, outcomes.clone())
                    }
                    TaskId::RemoveBare => {
                        let (s, m) = execute_remove_bare_task(&target_for_tasks);
                        (s, m, outcomes.clone())
                    }
                    _ => (
                        TaskStatus::Skipped,
                        TaskMessage::Ok("not applicable".into()),
                        outcomes.clone(),
                    ),
                }
            },
        );
    });

    let cwd = std::env::current_dir().unwrap_or_else(|_| PathBuf::from("."));
    let table = OperationTable::new(
        phases,
        worktree_infos,
        target.project_root.clone(),
        cwd,
        Stat::Summary,
        rx,
        TableConfig {
            columns: None,
            columns_explicit: false,
            sort_spec: None,
            extra_rows: 5 + (worktrees.len() as u16) * 4,
            verbosity: 0,
            pin_default_branch: false,
            partition_by_owner: false,
        },
        None,
    );
    let completed = table.run()?;

    orchestrator
        .join()
        .map_err(|_| anyhow::anyhow!("DAG orchestrator thread panicked"))?;

    if !completed.hook_summaries.is_empty() {
        eprintln!();
        eprintln!("Hooks:");
        for entry in &completed.hook_summaries {
            let status_word = if entry.warned { "warned" } else { "failed" };
            let exit_str = entry
                .exit_code
                .map(|c| format!("exit {c}"))
                .unwrap_or_else(|| "error".into());
            eprintln!(
                "  {}: {} {} ({}, {}ms)",
                entry.branch_name,
                entry.hook_type.filename(),
                status_word,
                exit_str,
                entry.duration.as_millis()
            );
            if let Some(ref out) = entry.output {
                for line in out.lines() {
                    eprintln!("    {line}");
                }
            }
        }
    }
    crate::commands::sync_shared::check_tui_failures(&completed.rows)?;
    Ok(())
}

         -path "${sandbox_dir}/state"  -prune -o \
         -name .git -print 2>/dev/null \

    exit 0
fi

for repo in "${repos[@]}"; do
    echo "Removing $repo"
    if ! daft repo remove --force "$repo"; then
        echo "  (failed — continuing)"
    fi
done

# `daft repo remove`

Removes a Git repository — bare directory plus every checked-out worktree —
running `worktree-pre-remove` and `worktree-post-remove` lifecycle hooks for
each worktree when the repo is daft-managed and trusted.

## Usage

    daft repo remove [<path>] [--force | -y] [--dry-run] [-v]

| Argument / flag | Description                                          |
| --------------- | ---------------------------------------------------- |
| `<path>`        | Repo path or any directory inside it (default: cwd). |
| `--force`, `-y` | Skip the interactive confirmation prompt.            |
| `--dry-run`     | Print the removal plan and exit without changes.     |
| `-v`            | Show hook details inline.                            |
| `-vv`           | Force the sequential (non-TUI) output path.          |

## Behavior

- Resolves the bare git directory via `git rev-parse --git-common-dir`. Refuses
  paths that are not inside a Git repository.
- Enumerates all checked-out worktrees via `git worktree list --porcelain`.
- For each worktree, runs `worktree-pre-remove` (if configured and trusted),
  removes the worktree, then runs `worktree-post-remove`.
- Hook failures **do not abort** the run. The repo is removed regardless; failed
  hooks appear in the post-run summary.
- After all worktrees are gone, removes the bare git directory and walks upward
  removing any now-empty parent directories. Drops the trust DB entry for the
  bare git path.
- If invoked from inside the removed repo, writes a safe target path to
  `DAFT_CD_FILE` so the shell wrapper `cd`s out of the deleted directory.

## Confirmation

By default, prompts before deletion. With `--force` (or `-y`), or in
non-interactive mode, the prompt is skipped. In a non-TTY context without
`--force`, the command exits with an error rather than proceeding silently.

## Examples

    # Remove the repo containing the current directory
    daft repo remove

    # Remove a repo by path
    daft repo remove ~/code/myproject

    # Preview what would happen
    daft repo remove --dry-run ~/code/myproject

    # Remove non-interactively
    daft repo remove --force ~/code/myproject
```
